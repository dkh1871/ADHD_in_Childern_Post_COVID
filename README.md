# ADHD Diagnoses in Children Post COVID-19  

## Description   

This is about investigating the ADHD diagnosis rates in children pre- and post-COVID-19. This project was about performing statistical analysis and practicing Exploratory Data Analysis (EDA) techniques. The idea for this project was spurred by the medication shortages for ADHD treatment medications that have occurred since the pandemic.  

## Data   
The data for this project come from the [National Health Interview Survey (NHIS)](https://www.cdc.gov/nchs/nhis/documentation/?CDC_AAref_Val=https://www.cdc.gov/nchs/nhis/data-questionnaires-documentation.htm) results from 2003 to 2025.  

Over the years, the format and questions asked in the survey have changed. This project will process data from each year and convert it so that the representation of the data is consistent with that of 2023.  

## Hypothesis  
There was no significant change in the post-COVID-19 period in the number of children diagnosed with ADHD.

## Techniques used  
* PMF  
* CDF  
* Chi2 testing  
* Regression
